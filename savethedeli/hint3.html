Hint 3 Helper — Character/Number Lookup + Encrypt/Decrypt

This helper uses the puzzle alphabet and mapping: A–Z plus space and period (28 symbols). It’s meant to make the arithmetic in Hint 3 quick to do by hand.

Alphabet mapping

A=0 B=1 C=2 D=3 E=4 F=5 G=6 H=7 I=8 J=9
K=10 L=11 M=12 N=13 O=14 P=15 Q=16 R=17 S=18 T=19
U=20 V=21 W=22 X=23 Y=24 Z=25 SPACE=26 PERIOD=27

Notes: Use a literal space character for SPACE. Period is ..

1) Numbers → Letters

Paste one or more space-separated numbers (0–27). This auto-converts them into characters as you type. Useful when you’ve computed values like key bytes and want to see symbols.

2) Letters → Numbers

Paste letters using only A–Z, space, and period. This auto-converts each character into its number (0–27). Useful for turning a crib like THE RECIPE. into numbers.

3) Encrypt: Pad numbers + Plaintext → Ciphertext

Enter pad numbers (0–27) and plaintext letters. This auto-computes: C = (P + K) mod 28. Pad must be at least as long as the plaintext.

4) Decrypt: Pad numbers + Ciphertext → Plaintext

Enter pad numbers (0–27) and ciphertext letters. This auto-computes: P = (C − K) mod 28. Pad must be at least as long as the ciphertext.

Tip: spaces are real characters. If your output looks “shifted,” check for missing/extra spaces.
